# Beispiel: Billo

Billo.fmp12 muss sich im selben Verzeichnis wie FX_Invoices.fmp12 befinden. **Billo** einfach öffnen und auf den *Erstellen*-Button klicken. 

Das Beispiel zeigt, wie einfach es ist, **FX  Invoices** zu implementieren: JSON und Rechnungs-PDF übergeben und das kombinierte PDF bzw. die XML-Rechnung übernehmen.

Wie man das JSON in einer Rechnungsdatei erstellt, zeigt das **Starter**-Beispiel.
