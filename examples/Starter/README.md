# Beispiel: Rechnungen

Rechnungen.fmp12 muss sich im selben Verzeichnis wie FX_Invoices.fmp12 befinden. **Rechnungen** einfach öffnen und auf *Rechnung erstellen* klicken. 

Die Startelösung Enthält die Tabellen Rechnungen, Positionen und zeigt, wie einfach es ist, **FX  Invoices** zu implementieren: JSON und Rechnungs-PDF übergeben und das kombinierte PDF bzw. die XML-Rechnung übernehmen.

Wie man das JSON in einer Rechnungsdatei erstellt, zeigt das **Starter**-Beispiel.
